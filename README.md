<h1><img src="assets/belo-horizonte-flag.png" alt="" height="30"> GPX BH</h1>

Ferramenta para gerar GPX de bairros de Belo Horizonte.

![](assets/gpx-bh.png)

**https://marcelochaves95.github.io/gpx-bh**

Escolha um bairro, visualize o limite no mapa e baixe o `.gpx`, abra no gpx.studio ou crie uma rota no Strava.

## Dados

Os limites dos bairros vêm do portal **[IDE-BHGEO](https://geonetwork.pbh.gov.br/geonetwork/srv/por/catalog.search#/metadata/db4df402-3e4c-4338-9c8b-d8e36c00ccce)** — Infraestrutura de Dados Espaciais da Prefeitura de Belo Horizonte (camada `ide_bhgeo:BAIRRO_POPULAR`).
